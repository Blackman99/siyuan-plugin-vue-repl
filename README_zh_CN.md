## 简介

一个帮助你在[思源笔记中](https://github.com/siyuan-note/siyuan)嵌入[Vue 交互式编辑器](https://github.com/vuejs/repl)的插件

## 特性
* 保留[Vue REPL](https://github.com/vuejs/repl) 的完整能力
* 自动保存 REPL 中的代码
* 自动跟踪挂件块大小
* 可离线使用

## 怎样使用

SiYuan -> 集市 -> 挂件 -> Vue REPL -> 安装
